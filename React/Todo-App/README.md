# Todo App

A modern React Todo application built using React and Tailwind CSS.

## Features

* Add Todo
* Delete Todo
* Edit Todo
* Mark Complete / Undo
* Filter Todos
* Local Storage Support
* Responsive UI

## Tech Stack

* React
* Tailwind CSS
* Vite

## Folder Structure

```bash
Todo-App/
│
├── public/
├── src/
├── package.json
└── vite.config.js
```

## Installation

```bash
npm install
npm run dev
```

## Learning Concepts

* React Components
* Props
* useState
* useEffect
* CRUD Operations
* Local Storage
* Component Architecture
