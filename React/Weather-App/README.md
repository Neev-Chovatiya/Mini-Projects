# Weather App 🌦️

A modern Weather Application built using React and Tailwind CSS.

This app fetches real-time weather data using the OpenWeather API and displays temperature, humidity, wind speed, weather condition, and dynamic UI updates based on weather conditions.

---

## Features

* Search weather by city
* Real-time weather data
* Temperature display
* Humidity and wind speed
* Dynamic weather icons
* Dynamic background based on weather
* Loading state handling
* Error handling
* Search using Enter key
* Responsive modern UI

---

## Tech Stack

* React
* Tailwind CSS
* Vite
* OpenWeather API
